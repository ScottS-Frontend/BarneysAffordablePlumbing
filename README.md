# Barney's Affordable Plumbing — Website Mockup

A single-page website mockup built as a sales concept for Barney's Affordable Plumbing, a family-owned plumbing business in Vacaville, CA (est. 1999).

**Live preview:** open `index.html` in any browser, or visit the GitHub Pages link once enabled (see below).

---

## About This Project

This is a **design mockup**, built to show Barney's Affordable Plumbing what a modern website could look like for their business. It's not yet a production-ready site — see [Known Limitations](#known-limitations--before-launch) below for what still needs to be done before it goes live.

## Design Direction

- **Concept:** A "work order ticket" theme throughout — reflecting the business's honest, no-surprises approach to pricing
- **Palette:** Deep workwear navy, oxidized copper, and warm paper tones (inspired by pipe materials and job-site ticket paperwork)
- **Type:** Condensed industrial display type paired with a monospace "ticket/receipt" accent for data-like details (phone numbers, service lists)

## Tech Stack

Plain HTML, CSS, and a small amount of vanilla JavaScript. No build tools, frameworks, or dependencies — just open `index.html` directly in a browser.

## Known Limitations / Before Launch

This mockup is **not production-ready**. Before it could go live as the real business site, it needs:

- [ ] **Real testimonials** — current reviews are placeholder text, not real customer quotes. Replace with actual, permissioned reviews.
- [ ] **Real photography** — the "about" section uses a styled graphic placeholder; a real job-site or team photo should replace it.
- [ ] **Working contact form** — the estimate request form is visual-only right now (no backend). Needs a form handler (e.g., Formspree, Netlify Forms, or a custom backend) to actually deliver submissions.
- [ ] **Domain** — needs to be registered and pointed at hosting (see notes on `barneysaffordableplumbing.com`).
- [ ] **Hosting** — needs a live host (static hosting is sufficient given the current tech stack).
- [ ] **Business email** — replace placeholder `info@barneysaffordableplumbing.com` with a real, monitored address.
- [ ] **License number** — add actual CA contractor's license number if the business wants it displayed.
- [ ] **Analytics** — consider adding Google Analytics or similar before launch to track traffic.
